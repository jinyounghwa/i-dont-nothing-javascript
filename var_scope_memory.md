  원시 값과 참조 감
ECMAscript의 변수는 원시값과 참조값에 데이터를 저장 할 수 있다. '원시값' 은 단순한 데이터이며 '참조값'은 여러 값으로 구성되는 객체이다.  
변수에 값을 할당하면 자바스크립트 엔진은 해당 값이 원시 데이터인지 참조 데이터인지 판단한다.
원시 타입은 undefined,null,불리언,숫자,문자열인데 이들 변수는 값으로 접근한다고 하며 실제 값을 조작한
